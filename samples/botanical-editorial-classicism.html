Editorial analysis

Reflective decks work best when the argument moves through three named layers

“Do not add beauty as ornament; make the structure itself feel composed.”
Context

Open with the social, institutional, or cultural reason the topic matters.

Mechanism

Explain the operating logic that turns intent into repeatable behavior.

Stewardship

Close with the decisions, rituals, and metrics that keep the system alive.

Editorial classicism sample02
Ecosystem map

Long-term strategy should map relationships, not decorate the page

People

Define who learns, decides, funds, governs, and benefits from the system.

Place

Show where the work happens and which routines make it durable.

System

Connect policy, data, incentives, evaluation, and feedback loops.

Acceptable visual move: labeled ecosystem map. Prohibited visual move: unrelated botanical line art in a corner.
