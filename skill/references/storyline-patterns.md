# Storyline Patterns

Use the pattern that best fits the deck intent.

## Executive Update

1. Executive summary
2. Context and decision needed
3. Current state
4. Key findings
5. Implications
6. Recommendation
7. Next steps

## Strategy Deck

1. Situation
2. Why now
3. Strategic choices
4. Evaluation criteria
5. Recommended direction
6. Operating model
7. Roadmap
8. Risks and mitigations
9. Decisions required

## Proposal Deck

1. Purpose
2. Client/business context
3. Challenges
4. Diagnosis
5. Options
6. Recommended approach
7. Workplan
8. Expected impact
9. Commercials or next steps

## Research Report

1. Summary
2. Scope and method
3. Market or issue overview
4. Findings
5. Evidence
6. Interpretation
7. Implications
8. Appendix

## AI / Tech Transformation

1. Why this matters now
2. Current capability gap
3. Use case portfolio
4. Value pool
5. Enablers
6. Operating model
7. Rollout roadmap
8. Governance and risk
9. Next actions

## Rule

Every section must advance the argument. Remove slides that only describe a topic without changing the reader's understanding.
